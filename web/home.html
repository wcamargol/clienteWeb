Free Liquid Web Template #001

Single Column Design

This web templates will adapt to the resolution of any viewing device. It will also expand and contract with the width of a visiting browser.

Know how to resize your browser window?

Grab the right side and squeeze it until it's just a skinny column.

Got cell phone internet service. Pull it up and watch it perform.

Note: This template has been modified, tested , passed as Mobile Friendly and scores 100/100 on Google PageSpeed Insights and A/A on GTmetrix.

Easy Construction

The HTML code used to build this template is simple and W3C compliant. In fact, we only used 13 HTML tags to complete the entire web page.

The embedded CSS (cascading style sheet) is also W3C compliant and we used the bare minimum to make it easier for beginners to understand.

You can use the template as a learning tool or use it to build your website.

This Template

This template demonstrates the method of displaying images without scaling.

Four generic images are provided. They are sized at 175, 250, 300 and 350 pixels wide.

Media queries are used to swap the images at different screen widths.

The stylesheet has been commented to show each step in the image swap process. (Remove all comments when you build your site for better performance.)

Media Queries

This template uses media queries to modify the appearance and structure of the lay out when viewed on smaller devices.

Margins and font sizes are changed for different screen widths using those media queries. They can also be used to swap different sized images.
